FIGI-USER INTERACTION

  • FIGI mediators are servers with fixed Internet connectivity.

  • A user connects to a FIGI server through a “Personal Information Roadmap” (PIR) Java-applet interface.

  • PIR is a personal financial information manager that can be used in three modes:

    • Profile registration and update: the user registers his interests with FIGI.
    • FIGI-session: the user connects and downloads collected information
    • Interactive: the user connects and retrieves information directly from information sources.
