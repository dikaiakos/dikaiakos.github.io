USER SERVICES IN FIGI

  • FIGI provides a number of available services :

    • Database access to collect financial data (stock prices, index values, exchange rates, etc.).
    • Retrieval and filtering of financial information from the Web.
    • Searching for WWW resources of interest.
    • Combined retrieval of database and semi-structured (Web) information.
    • Information alerts.

  • Each user adapts FIGI services to his own interests by annotating a “profile template.”
