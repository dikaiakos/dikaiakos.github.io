IMPLEMENTING FIGI

    FIGI is implemented using Java-based Mobile Agent technology:

    • Java is the programming language of choice for Internet programming and interfacing to Internet resources.
    • Mobile Agents are programs that can migrate from host to host in a wide-area network at times and to places of their own choice. The state of the running program is saved, transported to the new host and restored, allowing the program to continue where it left off.
    • Java-based Mobile Agents are Java objects that run in Java-based execution environments.
