USER PROFILES

  • The annotation of profile templates specifies:

    • the arguments of the queries that FIGI will execute for the user
    • the frequency of query execution
    • interest weights, notification priorities and alert mechanisms

  • The annotated profile templates represent the User profiles of FIGI, which are essentially sets of long-term, continuously evaluated queries.

  • A user profile incorporates user interests: stock quotes, analysis reports, news breaks, etc.
