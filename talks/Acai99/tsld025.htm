MITSUBISHI/s CONCORDIA PLATFORM

  • To implement FIGI/s architecture we used Mitsubishi/s CONCORDIA, a framework for the deployment of Mobile Agents.

  • CONCORDIA offers :

    • A platform for programming, launching and executing Java Mobile Agents.
    • Enhanced functionality for Agent synchronization, collaboration and messaging.
    • Service bridges (stationary agents)
    • Persistence, reliability, security.
