A PLAUSIBLE SCENARIO

    Users keep track of their portfolio and investment options by:

  • Checking email lists and newsgroups for investors. Storing some messages while deleting other.

  • Navigating through a number of “known” sites stored in a bookmarks file. “Cut and paste” interesting bits.

  • Using search engines - following unvisited links. Updating bookmarks.
