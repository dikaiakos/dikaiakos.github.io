Database Application Frameworks

  • Recent work has established that:

    • Java-based mobile agents represent a promising approach for providing database access over the Web.
    • Mobile agents offer many advantages over the typical applet-based approach for Web databases.
    • If not a “killer” mobile-agent application, surely a very promising one.

  • Database access over the Web can be materialized according the MA models presented earlier, leading to MA application frameworks for database applications.
