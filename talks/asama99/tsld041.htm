A Qualitative Comparison

  • Programmability: both platforms do a reasonably good job in providing classes and methods for developing mobile-agent applications.

  • Robustness: we experienced Aglet-router and, less often, Tahiti server failures. Aglets were more robust when running under AIX.

  • Interoperability with Java applets.

  • Mobility: both platforms support Weak Mobility.

  • Agent transfer protocols: RMI vs. ATP.

  • Documentation.
