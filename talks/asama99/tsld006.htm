Mobile-Agent Technologies

  • Mobile Agents: “programs that can migrate from host to host in a wide-area network at times and to places of their own choice. The state of the running program is saved, transported to the new host and restored, allowing the program to continue where it left off.”

  • Java-based mobile agents are Java objects that run in Java-enabled execution environments.

  • Mobile-agent application frameworks: libraries of mobile-agent routines implementing basic services according to some distributed-computing model.
