Remarks on Message Dispatch

  • Aglets Workbench is faster than Concordia when sending one message to another agent, or synchronizing with it via messaging.

  • With subsequent dispatches of the same message, however, Concordia’s average performance improves substantially.

  • Concordia/s performance over Aglets improves in message transmission from an applet and in message exchange between an applet and an agent. This is a side-effect of Aglets security model.
