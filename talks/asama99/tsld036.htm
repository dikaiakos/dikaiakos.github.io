Application Kernels

  • Consist of a short transaction (three queries) between a client and a remote database. Queries select all entries of a small database.

  • We measure time to:

      a) Launch an agent/message from the client site.

      b) Carry the agent/message to the database server.
      c) Connect to the database and execute the query.
      d) Bring the results back to the client.

  • Conduct two measurements: one for the first and another for subsequent queries (these are expected to be different).
